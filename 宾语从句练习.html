Practice Questions

1. I wonder _______ the meeting will be held tomorrow.

  • A. that
  • B. whether
  • C. what
  • D. how

Correct Answer: B

Use 'whether' for yes/no questions in object clauses, especially with 'or not' implied.

2. She asked me _______ I had finished my homework.

  • A. what
  • B. if
  • C. that
  • D. which

Correct Answer: B

Use 'if' for yes/no questions in object clauses when the main verb is in the past tense.

3. The teacher explained _______ the earth revolves around the sun.

  • A. that
  • B. if
  • C. what
  • D. whether

Correct Answer: A

Use 'that' for declarative object clauses, especially for objective truths, which take the present tense.

4. Can you tell me _______ the library is open today?

  • A. where
  • B. if
  • C. what
  • D. how

Correct Answer: B

Use 'if' for yes/no questions in object clauses with statement word order.

5. He didn’t know _______ caused the problem yesterday.

  • A. what

  • D. whether

Use 'what' as a connective pronoun for the content of the object clause.
